## 서비스 소개

### 서비스 개요
MOOD_ATLAS는 사용자의 감정에 맞춘 AI 수호자와의 대화를 통해 스트레스를 해소하고 감정을 관리하는 서비스입니다. 사용자는 포털에서 현재 감정을 입력하면, 4가지 수호자 타입(따뜻함, 행동, 통찰, 폭발) 중 자신에게 맞는 수호자를 선택해 AI 챗봇과의 깊이 있는 상담을 나눌 수 있습니다. 대화 후에는 감정 분석 결과와 보스 몬스터, 미션을 제공받으며, 미션을 완료함으로써 경험치를 얻고 수호자 친밀도를 높일 수 있는 게임화된 경험을 제공합니다.

### 타서비스와의 차별점
- **AI 기반 감정 분석**: 단순 상담을 넘어 LLM(올라마)을 통한 실제 감정 분석 및 개인화된 대응
- **게임화된 감정 관리**: 경험치, 레벨, 수호자 친밀도 등 게임 요소로 지속적 이용 동기 부여
- **다양한 수호자 타입**: 사용자의 성향에 맞춘 4가지 상담 스타일 제공
- **시각화된 감정 추적**: 각 대화 후 감정 통계와 보스 몬스터 도감으로 감정 변화 추적 가능

## 구현 내용 및 결과물

### 1. AI 수호자 상담 시스템
- 사용자가 선택한 수호자 타입에 맞춰 맞춤형 AI 상담 제공
- 실시간 채팅 인터페이스로 자연스러운 감정 표현 및 공감 경험
- LLM 기반 감정 분석으로 개인화된 조언 제공

### 2. 감정 분석 및 리포트 생성
- 대화 종료 후 자동으로 주요 감정, 감정 구성비 분석
- 감정에 해당하는 보스 몬스터 및 대처 미션 제시

### 3. 게임화 요소
- **경험치 시스템**: 미션 완료 시 경험치 획득 및 레벨업
- **수호자 친밀도**: 각 수호자와의 상담 횟수에 따른 친밀도 증가
- **도감 시스템**: 만난 보스 몬스터 및 수호자 정보 수집
- **프로필 시스템**: 현재 레벨, 경험치, EQ 스탯 조회

### 4. OAuth2 기반 로그인
- Google 소셜 로그인으로 간편한 회원가입 및 인증
- 사용자 정보 안전 관리

## 구현 방식

### 기술 스택
- **프론트엔드**: React + TypeScript (Vite)
- **백엔드**: Java Spring Boot
- **AI/LLM**: Python FastAPI + Ollama (로컬 LLM)
- **데이터베이스**: MySQL
- **인증**: OAuth2 (Google)

### 주요 아키텍처
```
FE (React) ←→ BE (Spring Boot) ←→ AI Server (Python FastAPI + Ollama)
```

---

### 설치 필수 사항
- Java 17+
- Node.js 18+
- Python 3.10+
- PostgreSQL 14+
- Ollama (로컬 LLM)

## 주요 기술 결정 사항

### OAuth2 Google 로그인 선택 이유
- 간편한 회원가입 및 로그인 사용자 경험 제공
- 개인정보 보안 강화
- 추가 인증 과정 없이 즉시 서비스 이용 가능

### LLM(Ollama) 선택 이유
- 로컬 환경에서 무료로 운영 가능
- 개인정보 보호 (클라우드 의존성 최소화)
- 배포 및 커스터마이징 용이

### Spring Boot + React 조합 선택 이유
- 안정적이고 성숙한 기술 스택
- 대규모 커뮤니티 및 풍부한 자료
- 향후 확장 및 유지보수 용이
